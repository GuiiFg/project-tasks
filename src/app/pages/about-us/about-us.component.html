Sobre Nós


O My Tasks é um projeto criado por uma pessoa só! Na sua contrução foi utilizado Angular no front-end, .Net/C# para o back-end e um banco de dados em PostgreSQL!

Você pode encontrar o código fonte no github, sinta-se a vontade para para usar como estudou ou fins educativos!

Front-End: https://github.com/GuiiFg/project-tasks

Back-End: https://github.com/GuiiFg/RestApiMyTasks

No banco de dados foram criadas duas tabelas, o sql usado para criar ambas estão o código do back-end em um arquivo chamado database.sql!

Todas as ilustrações presentes neste site foram retiradas da StotySet, deixo meu sincero agradecimento a ela pelo trabalho incrível deixando os projetos indies cada vez mais bonitos!
